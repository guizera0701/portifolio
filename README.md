# ✨ Landing Pages Premium

Uma experiência visual elegante que apresenta duas marcas distintas em uma única interface interativa:

- 🍝 **Italiana Food** — Restaurante com culinária italiana autêntica.
- 💈 **Barbearia Telasco** — Barbearia moderna focada em estilo e cuidado masculino.

## 🎯 Sobre o Projeto

Este projeto foi desenvolvido com foco em:

- Design moderno e sofisticado
- Experiência imersiva para o usuário
- Transições suaves
- Layout responsivo
- Navegação intuitiva
- Destaque visual para duas marcas independentes

A tela inicial divide a experiência em duas seções principais, permitindo que o visitante escolha qual ambiente deseja explorar.

## ⚙️ Como Executar Localmente

Clone o repositório:

```bash
git clone https://github.com/guizera0701/portifolio.git
```

Acesse a pasta:

```bash
cd portifolio
```

Abra o arquivo:

```bash
index.html
```

ou utilize uma extensão como **Live Server** no VS Code.
